\item colors\+: вектор, содержащий заранее заданные цвета.
\item current\+Index\+: индекс текущего цвета в векторе colors. Публичные методы класса\+:
\item Color\+Generator()\+: конструктор класса, инициализирует вектор colors и устанавливает начальное значение индекса current\+Index.
\item get\+Next\+Color()\+: возвращает следующий цвет из вектора colors и обновляет значение индекса current\+Index.
